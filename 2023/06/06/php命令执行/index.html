命令执行

函数

exec()

//command参数是要执行的命令
//output数组是保存输出结果
//return_var整形用来保存命令执行后的状态码，0代表执行成功，1代表执行失败

函数默认返回值是执行结果的最后一行，不会自动输出命令结构。

system()

//command参数是要执行的命令，
//return_var参数存放命令执行的状态码。

函数返回值也是执行结构的最后一行，它与exec()的不同是，它会自动输出命令结果。

passthru()

//command参数是要执行的命令，
//return_var参数存放返回的值，可不写该参数

它与system相同，都会自动输出命令结构，但是函数没有返回值。

shell_exec()

// cmd 要执行的命令
// 返回结果string

它没有记录状态码的参数，并且不能自动回显命令结果。但它会返回命令结果。其实shell_exec()只是``的一个变体，当shell_exec()不可用的时候，我们也不可以使用反引号。

popen()

$command是要执行的命令，另外一个是指针文件的连接模式mode，有r和w代表读和写。返回一个文件指针。

会对命令执行结果输出到到管道中。

proc_open()

类似popen,只不过更加灵活,它是打开一个双向通道。

pcntl_exec()

$path即命令执行路径，$envs即我们要执行的命令。

ob_start()(实验未成功)

调用时，输出缓冲区的内容会被当做参数去执行，并返回一个新的输出缓冲区作为结果，并被送到浏览器。
